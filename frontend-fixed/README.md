# Frontend (Admin + Farmer + Auth)

React + Vite + Tailwind + React Router + Axios + Lucide icons.

## Quick Start

```bash
cp .env.example .env
npm install
npm run dev
```

Open http://localhost:5173

## Configure backend URL
Edit `.env`:
```
VITE_API_BASE_URL=http://localhost:5050
```

## Build for production
```
npm run build
npm run preview
```
